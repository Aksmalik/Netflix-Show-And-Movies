🎬Netflix Content Analytics Dashboard

A dynamic, interactive data visualization project built to explore trends in movies and TV shows on Netflix — focusing on ratings, release trends, runtime patterns, and audience engagement over time.

⸻

2. Short Description / Purpose

The Netflix Content Analytics Dashboard is an interactive Power BI report designed to analyze historical trends in movies and TV shows, helping users understand how content production, ratings, and viewer engagement have evolved over the past several decades.

This dashboard is intended for data analysts, entertainment enthusiasts, and business stakeholders who want to explore patterns in content performance and audience preferences.

⸻

3. Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 Power BI Desktop – Main data visualization platform used to design and build the report.
* 📂 Power Query – Used for data cleaning, transformation, and shaping.
* 🧠 DAX (Data Analysis Expressions) – Created calculated measures like average IMDb score, total votes, and runtime metrics.
* 📝 Data Modeling – Structured relationships to enable filtering by year, content type, and certification.
* 📁 File Format – .pbix for development and .png for dashboard preview.

⸻

4. Data Source

Source: Public Netflix dataset (including movie and TV show metadata).

The dataset contains information such as:

	•	Title
	
	•	Release year
	
	•	Content type (Movie / TV Show)
	
	•	IMDb rating
	
	•	IMDb votes
	
	•	Runtime
	
	•	Age certification

This structured dataset enables analysis of content trends, audience engagement, and performance metrics across decades.

⸻

5. Features / Highlights

* Business Problem

With the rapid growth of streaming platforms, understanding what drives content popularity and how production trends change over time is crucial.

Key questions include:
	•	How has Netflix content volume grown over time?
	
	•	Do movies or TV shows have higher engagement?
	
	•	How do IMDb ratings vary across age certifications?
	
	•	What trends exist in runtime and viewer votes?

Raw datasets make it difficult to quickly uncover these insights without visualization.

⸻

* Goal of the Dashboard

To deliver an interactive analytical tool that:
	•	Enables exploration of Netflix content trends across decades
	
	•	Compares ratings, votes, and runtime patterns
	
	•	Helps identify audience engagement patterns
	
	•	Supports data-driven insights into content performance

⸻

* Walkthrough of Key Visuals

Year & Content Type Filters (Top Panel)
Interactive slicers allow users to filter data by release year range and content type (Movie or TV Show).

Top Rated Content (Bar Chart)
Highlights top movies and shows based on popularity and engagement metrics.

Skewness of Movies Over Time (Line Chart)
Shows distribution trends of movie releases across the last 20 years, highlighting growth patterns.

Average IMDb Score by Age Certification (Scatter Plot)
Compares how ratings differ across certification categories, helping identify audience preference trends.

Runtime Trends by Release Year (Bar + Line Chart)
Displays how average runtime has changed over decades, showing shifts in content format and storytelling style.

IMDb Votes & Ratings Trend (Dual Axis Chart)
Shows relationship between audience engagement (votes) and average ratings over time.

Total IMDb Votes by Year (Area Chart)
Highlights growth in audience engagement, especially in recent years with increased content consumption.

⸻

* Business Impact & Insights

Content Strategy Insights: Helps identify what type of content (movies vs shows) drives higher engagement.

Audience Preference Analysis: Reveals how ratings vary by certification and over time.

Trend Discovery: Shows significant growth in content production and viewer interaction in recent years.

Decision Support: Useful for analysts and media strategists to understand performance patterns and evolving viewer behavior.
